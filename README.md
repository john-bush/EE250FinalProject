Demo Link: https://drive.google.com/file/d/1Oya3b9_bXOIQKCyiEG-N0u2Mjk90H2Tg/view?usp=sharing 

Instructions: 

1. Connect servo motor and ultrasonic sensor to rpi. 
2. On VM, run `python3 vm_pub_and_sub.py`
3. On Raspberry Pi, run `python3 rpi_pub_and_sub.py`
4. You can now get close to the sensor which triggers the motor.
